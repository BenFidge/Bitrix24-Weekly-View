Booking Weekly View

Version 1.0.0

An alternative weekly view for the Booking module showing resources and their bookings in a grid layout with days across the top and resources down the left side.

Features

  • Weekly grid view with resources as rows and days as columns
  • Quick navigation between weeks using arrows or calendar picker
  • Respects your Bitrix24 locale settings (Sunday/Monday week start)
  • Click cells to create new bookings using native Bitrix24 dialogs
  • View booking details: time, customer name, phone number
  • Switch between daily and weekly views
  • Real-time updates when bookings change

Required Permissions

  • booking - Access to CRM Booking module (read/write)
  • user - Access to user settings and locale
  • placement - Register UI placement for menu
