# Resource types (categorize resources)

Official docs overview:
- https://apidocs.bitrix24.com/api-reference/booking/resource/resource-type/index.html

Resource types help categorize resources (e.g., Meeting Rooms, Trainers, Vehicles).

## Methods

- `booking.v1.resourceType.add`
- `booking.v1.resourceType.get`
- `booking.v1.resourceType.list`
- `booking.v1.resourceType.update`
- `booking.v1.resourceType.delete`
