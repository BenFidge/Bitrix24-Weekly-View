# Resources (what can be booked)

Official docs overview:
- https://apidocs.bitrix24.com/api-reference/booking/resource/index.html

A **resource** is the thing being booked (room, equipment, specialist, etc).

## Methods

### Create / Modify / Delete
- `booking.v1.resource.add` — add a new resource
- `booking.v1.resource.update` — update a resource
- `booking.v1.resource.delete` — delete a resource

### Read / List
- `booking.v1.resource.get` — retrieve a resource
- `booking.v1.resource.list` — list resources using a filter
